
# LMAO — AI 全球华人投资组合分析系统

本项目是“全球华人跨市场投资组合可视化分析 + AI 风险辅助决策平台”的 MVP 工程骨架，涵盖：

- FastAPI 后端服务
- MySQL 数据库结构
- AI Prompt 模板
- Flutter 前端结构

## 模块结构

```
backend/   # FastAPI + 数据模型 + API
db/        # MySQL DDL
ai/        # AI Prompt 模板
frontend/  # Flutter UI 结构
```

## 合规说明

- 本系统不提供投资建议，仅提供数据分析与风险提示  
- 所有 AI 输出都包含风险提示与免责声明

## 适用人群

- 全球华人跨市场投资者
- 机构/团队内部投资组合分析
